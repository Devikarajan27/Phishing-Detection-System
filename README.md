# 🔐 Phishing Detection System

A Machine Learning based web application that detects whether a website URL is legitimate or phishing.

## 🚀 Features
- Machine Learning Model
- Flask Web App
- Confidence Percentage
- Interactive Chart Visualization

## 🛠 Technologies Used
- Python
- Scikit-learn
- Flask
- Pandas
- NumPy

## ▶ How to Run

1. Install dependencies:
   python -m pip install -r requirements.txt

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000
